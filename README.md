# Elite Computadores

Site institucional e catálogo premium da Elite Computadores, especializado em computadores gamer de alta performance, setups personalizados e periféricos topo de linha.

## 🚀 Estrutura do Projeto

O site é totalmente estático e construído de forma nativa com tecnologias padrão de web:
- **HTML5** (Estrutura semântica e SEO nativo)
- **Vanilla CSS** (Design responsivo de alta fidelidade "Gamer Premium" com transições fluidas)
- **Vanilla Javascript** (Starfield interativo de alto desempenho, animações de scroll e timeline de rastreio de pedido)
